Репозиторий (Repository): Место, где хранятся файлы проекта и история изменений.
Коммиты (Commits): Снимки состояния файлов.
Состояния файлов:

Working Directory (Рабочий каталог)
Staging Area (Индексация)
Repository (Репозиторий)

Основные команды Git:

git init: Инициализация нового репозитория.

git clone [url]: Клонирование удалённого репозитория.

git status: Проверка текущего состояния.

git add [file]: Добавление файлов для коммита.

git commit -m "message": Создание коммита.

git push: Отправка коммитов на удалённый репозиторий.

git pull: Получение изменений из удалённого репозитория.
